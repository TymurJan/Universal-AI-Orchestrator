# Security Policy

## Supported Versions

Only the latest version of Universal AI Orchestrator is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our governance systems very seriously. If you find a vulnerability, please do NOT open a public issue. 

Instead, please report it privately by emailing: **ngo.talan.ua@gmail.com**

Our security team (Antigravity Support) will acknowledge your report within 48 hours and provide a timeline for a fix.

## 🛡️ Commercial Core
Users of the **Commercial Core** receive priority security audits and automated patches. If you are using the obfuscated version, ensure your license is active to receive encrypted security modules.
