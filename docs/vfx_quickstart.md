# 🚀 Швидкий Старт: Кінематографічна Автоматизація

Ви все налаштували для того, щоб перетворити лендінг на голлівудський проект. Ось ваша інструкція: що, куди і як «княпати».

## Крок 1: Отримання API-ключа (Links)

Оберіть один або кілька сервісів (я рекомендую почати з Luma для серця та Runway для щита):

*   **Luma Dream Machine API:** [lumalabs.ai/dream-machine/api](https://lumalabs.ai/dream-machine/api) — найкраще для реалістичних рухів.
*   **Runway Gen-3 API:** [runwayml.com](https://runwayml.com/) — фаворит для енергетичних ефектів.
*   **Kling AI:** [klingai.com](https://klingai.com/) — для найтриваліших та найскладніших сцен.

> [!TIP]
> Після реєстрації шукайте розділ **API Keys** або **Developer Settings** у вашому кабінеті. Скопіюйте ключ, він схожий на довгий набір символів (наприклад: `luma-sk-xxxx...`).

## Крок 2: Куди вставити (The Key)

Відкрийте файл `.env` у корені проекту (якщо його немає, створіть порожній файл з такою назвою) і додайте туди ваш ключ:

```env
LUMAAI_API_KEY=ваш_ключ_тут
RUNWAYML_API_SECRET=ваш_ключ_тут
```

## Крок 3: Як запустити (The Command)

Відкрийте термінал (CMD або PowerShell) у папці проекту і введіть одну команду:

```bash
python generate_cinematic_vfx.py
```

Цей скрипт зробить все за вас: він сам підготує промпти і запустить генерацію.

## Крок 4: Фінальна заміна (The Result)

Після того, як ви отримаєте відео-файл (наприклад, `heart_motion.webm`):
1.  Покладіть його в папку `landing/`.
2.  В `index.html` я вже підготував місця для вставки. Вам треба буде просто замінити `<img>` на `<video>`. Але не хвилюйтеся — як тільки у вас буде файл, **просто скажіть мені**, і я сам оновлю код за секунду.
