# 🚀 Інструкція зі встановлення та запуску

Ваш продукт тепер — це не просто опис, а реальний інструмент. Ось як його розгорнути за 1 хвилину:

## 1. Встановлення залежностей
Переконайтеся, що у вас встановлено Python 3.9+, та виконайте команду:
```bash
pip install -r requirements.txt
```

## 2. Налаштування середовища (Optional)
Створіть файл `.env` у корені проекту та додайте ключі (якщо плануєте реальне підключення до API):
```env
ANTHROPIC_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here
```

## 3. Запуск Аудиту та Оркестрації
Просто запустіть головний скрипт:
```bash
python orchestrator.py
```

### Що відбудеться після запуску:
1. **Deep Scan:** Система реально перевірить усі файли проекту на наявність секретів (API ключів) та логічних конфліктів.
2. **Orchestra Demo:** Запуститься ланцюжок агентів (Analyst -> Optimizer), де результат першого автоматично передається другому через **State Management**.
3. **Report:** У папці `REPORTS/` з'явиться детальний Markdown-звіт про стан вашого проекту.
